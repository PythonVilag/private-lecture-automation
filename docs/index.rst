Private Lecture Automation
==========================

**Private Lecture Automation** contains tools that make managing private lectures easier.

These tools currently include:

- Sending out an introduction email to new potential clients (``send_introduction_email``)
- Sending a calendar event for the next lecture of a client (``send_calendar_event``)


Check out the :doc:`installation` section for further information.
